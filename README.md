# Pixeld++

 [Pixeld](https://github.com/vincetse/pixeld) was built several years ago as an exercise to learn TypeScript.  Fast forward to 2026, I decided to port Pixeld to C++ using Google Gemini to do the heavylifting.  Pixeld++ is the outcome of the exercise.
